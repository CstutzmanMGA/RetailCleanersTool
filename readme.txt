# RetailCleanersTool

## Description
This is intended to allow for the searching of available cleaning supplies using apis and potential in person store data and compare the cost per volume to give an idea of what cleaning supplies are most cost effective for bulk use:

- It will look for Walmart, Target, Amazon, and other places to give cost comparison.
- This helps people that use cleaners regularly reduce cost with a real time comparison
- Short term rental owners, and cleaners.

---

## Features

This will compare any cleaner that it can find on the web to a know cleaning supply list, and any manually inputted items to give a good comparison of those cleaning supplies.-
